Export Certificate / Preserved Record 情報

官公庁様式の書類用フィールドです。車検証・輸出抹消仮登録証明書の内容を転記してください。

PDFまたは写真から項目を自動抽出します。

入金明細

写真

車両の写真を複数枚アップロード。ドラッグ&ドロップ可。クリックで拡大表示、マウスオーバーで削除ボタン表示。

書類ごとの Buyer / 条件

書類タイプごとに異なるBuyerを指定できます。未指定の書類はスキップされます。

Case Code 自動採番

新規案件作成時に Case Code を自動生成するパターン。空欄の場合は自動採番しません。

使えるプレースホルダー:
{###} / {####} — 連番（桁数指定、ゼロ埋め）
{YYYY} — 西暦4桁、{YY} — 西暦2桁
{MM} — 月、{DD} — 日
例: RAYA_{###} → RAYA_001、{YY}{MM}-{##} → 2510-01

Invoice Ref No 自動採番

新規案件作成時に Invoice Ref No を自動生成するパターン。

会社ロゴ

書類の社名の上に表示されます。PNG/JPG/SVG形式、推奨幅300px。

ロゴ未設定

署名者情報

書類のフッターに表示される署名者名・肩書です。

定型文カスタマイズ

書類に印字される条項・注記を編集できます。空欄でデフォルト値を使用。

ブラウザ通知

船積予定・入金期日が近づいたときにブラウザ通知でお知らせします。

メール下書き

案件の書類プレビュー画面から、Buyer宛のメール下書きを生成できます。
